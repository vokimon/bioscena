Agent
	- Nom

MultiAgent
	- string: Nom
	- list<CAgent*>: Llista ordenada d'agents subordinats
	- Exemple de configuracio texte

		Agents2: 
			Tipus(Agent/Multiple) 
			Agent(Posicionador1)
			Agent(Posicionador2)
			Agent(Direccionador1);

Temporitzador
	- string: Nom
	- Cicle Actiu: Determina quan triguen els periodes de temps actius
		- uint32: Periode minim
		- uint32: Numero de daus
		- uint32: Magnitud dels daus (Van de zero a la magnitud)
	- Cicle Inactiu: Determina quan triguen els periodes de temps inactius
		- uint32: Periode minim
		- uint32: Numero de daus
		- uint32: Magnitud dels daus (Van de zero a la magnitud)
	- Agents subordinats
		- list<CAgent*>: Llista ordenada de subordinats que s'executen en el cicle actiu
		- CAgent*: Agent de antiaccio (S'execute en cicle inactiu)
	- Cicle actual (Valors del temporitzador quan es reemprengui la marxa)
		- bool: Cicle actiu o inactiu
		- uint32: Periode restant del cicle actual
	- Exemple de configuracio texte

		Temporitzador1
			Tipus(Agent/Multiple/Temporitzador)
			Agent(Posicionador3);
			CicleActiu(34,2,5) 
			CicleInactiu()
			CicleActual(3, true)
Iterador
	- string: Nom
	- Iteracions: Determina quantes vegades es repiteixen els subordinats
		- uint32: Iteracions minimes
		- uint32: Numero de daus
		- uint32: Magnitud dels daus (Van de zero a la magnitud)
	- Agents subordinats
		- list<CAgent*>: Llista ordenada d'agents subordinats
		- CAgent*: Agent de preaccio (S'executa un sol cop abans de tot)
		- CAgent*: Agent de postaccio (S'executa un sol cop despres de tot)
	- Exemple de configuracio texte

		Iterador3
			Tipus(Agent/Multiple/Iterador)
			Agent(Posicionador3);
			Probabilitat(20,100) 

Aleaturitzador
	- string: Nom
	- Probabilitat d'activacio
		- uint32: Numero de intents o mostra
		- uint32: Numero d'encerts que segons la probabilitat tenderien a donar-se
	- Agents subordinats
		- CAgent*: Agent d'accio (S'execute si es dona la probabilitat)
		- CAgent*: Agent de reaccio (S'execute si no es dona la probabilitat)
	- Exemple de configuracio texte

		Aleaturitzador1
			Tipus(Agent/Multiple/Aleaturitzador)
			Agent(Posicionador3);
			Probabilitat(20,100)

PosicionadorZonal

	- uint32: Posicio actual
		* Han d'existir a la topologia
	- CPosicionador*: Agent Posicionador central (dependencia)
